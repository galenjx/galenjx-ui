# galenjx-ui

这是我学习封装vue组件的记录

## 现已支持

Button（按钮）、Radio（单选框）、Checkbox（复选框）、Input（输入框）、Switch（滑动）、Form（表单域）、Dialog（对话框）

## 安装
```shell

 npm i galenjx-ui --save

```

## 引入 galenjx-ui,css文件，然后注册就可以在组件中使用了

```js

import GalenjxUi from 'galenjx-ui'
Vue.use(GalenjxUi)
import "galenjx-ui/dist/galenjx-ui.css"

```
